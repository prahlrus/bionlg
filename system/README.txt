###

This is a natural language generation system for creating biographical summaries from RDF files encoding the Chinese Biographical database. It is copyright December 2012 by William Prahl and John David Stone. 

To use the system, enter the following into a shell window:

./bio_nlg DATA_PATH

Where DATA_PATH is a path to the directory containing the registry of the named entities and the files describing them.
